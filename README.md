# Virtualcode BOGO Same Product (WooCommerce)

**Buy One Get One Free (BOGO) Same Product for WooCommerce** is a lightweight WordPress plugin that helps store owners run BOGO promotions easily. When an eligible product is added to the cart, the quantity is automatically doubled while the customer only pays for one. The plugin supports simple, variable, and grouped products and respects sale pricing.
